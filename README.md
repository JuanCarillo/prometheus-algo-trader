# Prometheus Algo Trader

A multi-asset algorithmic trading system operating across crypto and equities, built to autonomously manage execution, risk, and capital allocation across multiple strategies.

Designed for continuous operation with real-time market adaptation, fault tolerance, and full lifecycle trade management.


## Key Capabilities

- Multi-asset automation across crypto and equities  
- Real-time market regime detection and adaptive strategy execution  
- Dynamic capital allocation and risk-controlled position sizing  
- Fully automated trade lifecycle management (entry → scaling → exit)  
- Fault-tolerant system with crash recovery and state persistence

- ## Status

Currently running in live paper trading mode for validation and performance benchmarking. Transition to full capital deployment is gated by risk-adjusted performance thresholds.

## Why This Project

This system demonstrates the ability to design and deploy complex, multi-component automation pipelines involving:

- Real-time data processing  
- Decision systems under uncertainty  
- Risk management and capital allocation  
- System reliability and recovery  

It reflects practical experience building production-like systems beyond simple scripts or backtests.
